## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Worktree merge with same untracked file in main | When the planner agent writes a PLAN.md to the main tree before the executor's worktree branch is merged, both have the file and `git merge` aborts with "untracked working tree files would be overwritten." This wasted 15 mins recovering via reflog. | advisory | Before merging an executor worktree, `rm -f` any untracked file in the main tree that is also tracked on the worktree branch. Mockup HTMLs cost minutes; rewrites cost hours. |
